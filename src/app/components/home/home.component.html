HelpDesk

Olá! Seja bem-vindo ao sistema HelpDesk.
Este é um sistema para gerenciamento de chamados que simula um simples Help Desk desenvolvido em Angular 12 e Spring Boot 3.
O sistema está rodando na nuvem, com o frontend hospedado na plataforma Vercel e o backend na Azure. O projeto tem como objetivo principal abordar os conceitos de aplicações SPA, autenticação e autorização utilizando tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.
